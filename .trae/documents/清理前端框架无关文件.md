# 前端框架无关文件清理计划

## 项目结构分析

* 基于 Vue-Vben-Admin 框架的前端项目

* 主要应用：`apps/web-antd`（构建输出：`@chet/chet.quartznet.ui`）

* 使用 pnpm + turbo 构建工具

## 清理范围

### 1. 构建缓存与临时文件

* **删除**：所有 `node_modules` 目录（依赖会在构建时重新安装）

* **删除**：所有 `.turbo` 目录（turbo 构建缓存）

* **删除**：所有 `.vite` 和 `.vite-temp` 目录（vite 构建缓存）

* **删除**：所有 `.vue-global-types` 目录（Vue 全局类型缓存）

### 2. 未使用的应用与示例

* **保留**：`apps/web-antd`（唯一使用的应用）

* **删除**：其他未使用的应用目录（如果存在）

### 3. 未使用的框架组件

* **检查**：`packages` 目录中的包，只保留项目实际使用的

* **删除**：未在 `web-antd` 中使用的包

* **检查**：只保留 antd 相关样式，删除其他 UI 框架样式（如 ele、naive）

### 4. 测试与文档文件

* **删除**：所有 `__tests__` 目录

* **删除**：框架自带的示例文档

* **保留**：项目必要的 README 和配置文档

### 5. 冗余配置与脚本

* **检查**：`scripts` 目录，只保留必要的构建脚本

* **删除**：未使用的 lint、check、format 等脚本

* **简化**：`package.json` 中的冗余脚本和依赖

## 清理步骤

1. **清理构建缓存**：使用 `pnpm clean` 命令或直接删除缓存目录
2. **移除未使用的应用**：删除除 `web-antd` 外的其他应用
3. **精简 packages**：分析并删除未使用的包
4. **清理样式**：只保留 antd 相关样式
5. **移除测试文件**：删除所有 `__tests__` 目录
6. **精简脚本和依赖**：更新 `package.json`
7. **验证构建**：运行 `pnpm run build:antd` 确保清理后能正常构建

## 预期效果

* 减少项目体积，提高构建速度

* 去除与项目无关的框架冗余文件

* 保持项目结构清晰，只保留必要的代码和配置

* 确保构建流程正常运行

## 注意事项

* 清理前备份重要文件

* 确保删除的文件确实与项目无关

* 清理后验证构建和运行正常

* 遵循最小化原则，只保留必要的功能和配置
