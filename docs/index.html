📦 快速安装

# 安装核心包 dotnet add package Chet.QuartzNet.UI # 根据需求安装数据库驱动 dotnet add package Chet.QuartzNet.EFCore.SQLite

🚀 简单三步，即可运行

  • 1

    注入服务

    在 Program.cs 中调用 AddQuartzUI 注入依赖。

  • 2

    启用中间件

    使用 UseQuartz 启用 UI 路由与静态资源。

  • 3

    访问管理页

    启动应用，访问 /quartz-ui 即可开始管理。

Program.cs C# 
builder.Services.AddQuartzUI(config);
builder.Services.AddQuartzClassJobs();

var app = builder.Build();
app.UseQuartz();
app.Run();
