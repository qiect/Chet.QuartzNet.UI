# 🔥 Chet.QuartzNet.UI 通知管理攻略：多渠道触达+实时响应！（todo）

## 🎯 什么是通知管理功能？

通知管理功能是 Chet.QuartzNet.UI 的重要功能之一，它允许你在作业执行成功、失败或系统发生异常时，通过多种渠道接收通知。这样你可以及时了解系统状态，快速响应问题，确保任务调度的可靠性！😍

## ✨ 通知管理有哪些核心功能？

### 📣 多种通知渠道
- **微信通知**：通过 PushPlus 发送微信消息
- **企业微信通知**：发送到企业微信群或个人
- **钉钉通知**：发送到钉钉群或个人
- **邮件通知**：发送到指定邮箱
- **短信通知**：发送到手机（需配置短信服务商）

### 🎯 灵活的通知触发条件
- **作业执行成功**：当作业执行成功时发送通知
- **作业执行失败**：当作业执行失败时发送通知
- **调度器异常**：当调度器发生异常时发送通知
- **作业超时**：当作业执行时间超过设定阈值时发送通知

### 📝 自定义通知模板
- 支持自定义通知标题和内容
- 可以包含作业名称、执行时间、执行结果等动态变量
- 支持 Markdown 格式，美化通知内容

### 🔧 通知配置管理
- 可以配置多个通知渠道
- 支持为不同作业设置不同的通知方式
- 可以启用或禁用特定的通知渠道

### 📊 通知发送记录
- 记录所有发送的通知
- 显示通知的发送时间、接收人、状态等信息
- 支持查看通知的详细内容

## 🚀 如何配置和管理通知？

### 1️⃣ 登录管理界面

访问 `http://localhost:port/quartz-ui`，使用用户名和密码登录。

### 2️⃣ 进入通知管理页面

点击左侧菜单的「通知管理」，进入通知配置页面。

### 3️⃣ 配置通知渠道

#### 📱 微信通知配置

1. 点击「新增通知渠道」按钮
2. 选择「微信通知」类型
3. 填写 PushPlus Token（需要在 PushPlus 官网注册获取）
4. 选择通知模板
5. 点击「保存」按钮

#### 🏢 企业微信通知配置

1. 点击「新增通知渠道」按钮
2. 选择「企业微信通知」类型
3. 填写企业微信 CorpId、AgentId 和 Secret
4. 填写接收消息的用户或部门
5. 选择通知模板
6. 点击「保存」按钮

#### 📞 钉钉通知配置

1. 点击「新增通知渠道」按钮
2. 选择「钉钉通知」类型
3. 填写钉钉机器人 Webhook URL
4. 设置安全设置（可选）
5. 选择通知模板
6. 点击「保存」按钮

#### 📧 邮件通知配置

1. 点击「新增通知渠道」按钮
2. 选择「邮件通知」类型
3. 填写 SMTP 服务器信息
4. 填写发件人邮箱和密码
5. 填写收件人邮箱
6. 选择通知模板
7. 点击「保存」按钮

### 4️⃣ 为作业配置通知

#### 📝 在新增作业时配置

1. 在新增作业页面，找到「通知设置」部分
2. 选择要启用的通知渠道
3. 设置通知触发条件
4. 点击「保存」按钮

#### 🔄 在作业编辑时配置

1. 进入作业管理页面，找到要编辑的作业
2. 点击「编辑」按钮
3. 修改「通知设置」部分
4. 点击「保存」按钮完成修改

### 5️⃣ 查看通知发送记录

1. 进入通知管理页面
2. 点击「通知记录」标签页
3. 查看所有发送的通知记录
4. 可以按时间范围、通知类型、发送状态进行筛选
5. 点击「详情」按钮查看通知的详细内容

## 💡 通知管理的最佳实践

### 🎯 选择合适的通知渠道
- 根据团队的沟通习惯选择通知渠道
- 关键作业可以配置多种通知渠道，确保消息触达
- 对于不同级别的通知，选择不同的渠道

### ⚡ 合理设置通知触发条件
- 对于关键作业，同时启用成功和失败通知
- 对于非关键作业，只启用失败通知
- 避免设置过多的通知，防止通知泛滥

### 📝 优化通知模板
- 通知内容要简洁明了，包含关键信息
- 使用动态变量，提供作业的具体信息
- 对于失败通知，包含异常信息的摘要

### 🔧 定期检查通知配置
- 定期测试通知渠道是否正常工作
- 及时更新过期的 Token 或密码
- 确保接收人信息的准确性

### 📊 分析通知记录
- 定期查看通知发送记录
- 分析通知的发送频率和失败率
- 优化通知配置，提高通知的有效性
