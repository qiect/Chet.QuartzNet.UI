# 🔥 Chet.QuartzNet.UI 日志管理指南：快速排查故障+数据分析！（todo）

## 🎯 什么是日志管理功能？

日志管理功能是 Chet.QuartzNet.UI 的重要功能之一，它记录了所有作业的执行过程和结果。通过查看日志，你可以了解作业的执行情况、排查问题、优化性能，是系统维护和故障诊断的重要工具！😍

## ✨ 日志管理有哪些核心功能？

### 📝 日志记录
- 记录作业的开始时间和结束时间
- 记录作业的执行结果（成功/失败）
- 记录作业执行过程中的输出信息
- 记录作业执行异常的详细堆栈信息

### 🔍 日志查询
- 支持按作业名称、作业组、执行状态进行筛选
- 支持按时间范围查询
- 提供全文搜索功能，搜索日志内容
- 支持分页查看大量日志

### 📊 日志分析
- 统计作业的执行次数和成功率
- 计算作业的平均执行时间
- 显示作业执行状态分布
- 提供执行趋势图表

### 💾 日志导出
- 支持将日志导出为 Excel 或 CSV 格式
- 可以选择导出的时间范围和作业
- 便于离线分析和报告生成

### 🗑️ 日志清理
- 支持自动清理旧日志
- 可以设置日志保留天数
- 避免日志过大影响系统性能

## 🚀 如何管理日志？

### 1️⃣ 登录管理界面

访问 `http://localhost:port/quartz-ui`，使用用户名和密码登录。

### 2️⃣ 进入日志管理页面

点击左侧菜单的「日志管理」，进入日志列表页面。

### 3️⃣ 查看日志列表

在日志列表页面，你可以看到所有作业的执行日志，包括作业名称、组、执行状态、开始时间、结束时间等信息。

### 4️⃣ 筛选和搜索日志

#### 📅 按时间范围筛选
- 选择「今日」、「本周」、「本月」或「自定义时间范围」
- 点击「查询」按钮查看指定时间范围内的日志

#### 🔍 按作业筛选
- 选择作业名称或作业组
- 可以多选作业进行筛选

#### ⚡ 按状态筛选
- 选择「全部」、「成功」或「失败」
- 快速查看特定状态的日志

#### 🔤 全文搜索
- 在搜索框中输入关键词
- 系统将搜索日志内容中包含该关键词的记录

### 5️⃣ 查看日志详情

点击日志记录的「详情」按钮，进入日志详情页面，查看完整的日志信息，包括：
- 作业基本信息
- 执行时间和耗时
- 执行结果
- 输出信息
- 异常堆栈信息（如果执行失败）

### 6️⃣ 导出日志

1. 设置筛选条件，选择要导出的日志范围
2. 点击「导出」按钮
3. 选择导出格式（Excel/CSV）
4. 保存导出文件到本地

### 7️⃣ 清理日志

1. 点击「清理日志」按钮
2. 设置日志保留天数（如保留30天）
3. 点击「确认清理」按钮
4. 系统将自动清理超过保留天数的日志

## 💡 日志管理的最佳实践

### 🎯 日志查询技巧
- 结合多种筛选条件，快速定位问题日志
- 使用时间范围筛选缩小查询范围
- 对于失败的作业，重点查看异常堆栈信息

### 📊 日志分析方法
- 定期查看作业的执行统计，发现异常趋势
- 分析作业的平均执行时间，优化性能
- 关注失败率高的作业，及时排查问题

### 💾 日志保留策略
- 根据系统规模和业务需求，设置合理的日志保留天数
- 对于关键作业，可以延长日志保留时间
- 定期导出重要日志，进行离线备份

### 🔧 故障诊断流程
1. 查看失败的日志记录
2. 分析异常堆栈信息，定位问题原因
3. 检查作业配置和调度表达式
4. 测试作业执行，验证修复效果
5. 更新作业配置或代码，解决问题

### 📈 性能优化建议
- 分析执行时间长的作业，优化业务逻辑
- 避免在作业中执行耗时过长的操作
- 对于频繁执行的作业，考虑异步处理
