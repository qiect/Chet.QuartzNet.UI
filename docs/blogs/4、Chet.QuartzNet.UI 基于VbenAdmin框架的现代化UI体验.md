## 👨💻 如何使用这些UI功能？

### 1. 主题切换

1. 点击页面右上角的「设置」按钮
2. 在「外观设置」中选择你喜欢的主题模式
3. 可以选择预设主题颜色或自定义主色调
4. 实时预览主题效果

### 2. 语言切换

1. 点击页面右上角的「语言」按钮
2. 在下拉菜单中选择「中文」或「English」
3. 页面语言会立即切换，无需刷新

### 3. 偏好设置

1. 点击页面右上角的「设置」按钮
2. 在「布局设置」中选择你喜欢的布局方式
3. 在「菜单设置」中调整菜单显示方式
4. 在「顶部设置」中调整顶部栏显示内容

### 4. 使用快捷键

1. 点击页面右上角的「设置」按钮
2. 选择「快捷键」查看所有支持的快捷键
3. 在日常使用中使用这些快捷键，提升操作效率

### 5. 通用设置

1. 点击页面右上角的「设置」按钮
2. 在「通用设置」中调整水印、页面切换动画等
3. 实时预览设置效果
